# Neon Drift X

Более насыщенная браузерная аркада с уровнями сложности и несколькими игровыми механиками.

## Что добавлено

- 3 сложности: `Easy`, `Normal`, `Hard` (клавиши `1/2/3`)
- Очки здоровья (HP)
- Комбо-множитель за серию сбора звёзд
- Пауэрапы:
  - `S` (Shield) — щит, поглощает урон
  - `T` (Time Slow) — замедляет время
- Рывок (`Shift`) с неуязвимостью и кулдауном
- Пауза (`P`)
- Сохранение лучшего результата в `localStorage`

## Запуск

Открой `index.html` в браузере или подними локальный сервер:

```bash
python3 -m http.server 8000
```

Затем открой `http://localhost:8000`.

## Управление

- `←` / `→` или `A` / `D` — движение
- `Shift` — рывок
- `P` — пауза
- `1` / `2` / `3` — выбор сложности
- `Space` — старт / рестарт
