# Neon Drift X+

Аркадная браузерная игра с несколькими уровнями сложности и расширенными механиками прогрессии.

## Новые механики

- 3 сложности: `Easy`, `Normal`, `Hard` (клавиши `1/2/3`)
- Волны сложности: со временем растёт темп и шанс элитных угроз
- Типы угроз:
  - обычные/быстрые/элитные метеоры
  - кометы с горизонтальным пролётом
- Пауэрапы:
  - `S` — щит
  - `T` — замедление времени
  - `H` — ремонт HP
- Комбо за сбор звёзд
- Рывок (`Shift`) с кулдауном
- EMP-способность (`E`) на 100% энергии, очищающая область
- Кристаллы как валюта прогрессии
- Апгрейд между забегами: `U` (+1 к максимальному HP, стоимость растёт)
- Сохранение рекорда, кристаллов и апгрейда через `localStorage`

## Запуск

```bash
python3 -m http.server 8000
```

Открой `http://localhost:8000`.

## Управление

- `←` / `→` или `A` / `D` — движение
- `Shift` — рывок
- `E` — EMP при полной энергии
- `P` — пауза
- `1` / `2` / `3` — выбор сложности
- `U` — апгрейд корпуса между забегами
- `Space` — старт / рестарт
## Если в веб-редакторе не активна кнопка Commit

Обычно это значит, что нет новых изменений или остались маркеры конфликта (`<<<<<<<`, `=======`, `>>>>>>>`).
В этой версии файлов конфликтных маркеров нет — можно коммитить изменения после любого правочного шага.
