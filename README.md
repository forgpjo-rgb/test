# Neon Drift

Небольшая браузерная аркада: управляй неоновым кораблём, уворачивайся от метеоров и собирай звёзды.

## Запуск

Открой `index.html` в браузере или запусти локальный сервер:

```bash
python3 -m http.server 8000
```

После этого перейди на `http://localhost:8000`.

## Управление

- `←` / `→`
- `A` / `D`
- `Space` — старт и рестарт
